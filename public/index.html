Untangled 2016

▽

Overview

We want to inspire young people to creative coding and related efforts for a week this summer.

Theme

We’ll find up a selection of disconnected items: appliances, tissue boxes, fans, toasters etc… item(s). We’ll invite votes to see what gets internetized, and then we’ll team up:

  • a design and UX creative group to build a web interface to control the connected hardware
  • a hardware group to attach sufficient tech goods to the junk to allow control over the internet
  • a programming group to wire up the web interface to the hardware

Requirements

  • Age (13-18)

    We'll flex this if a younger programmer can demonstrate coding experience; and we know they're out there.

  • Desire to learn programming, hardware or design

    The week will be about learning how to write computer code, use computer hardware and designing computer based user interfaces

  • Dedication

    Untangled is for a week, not just a day
